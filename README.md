# Orbit API

This repo contains the API for [Orbit](https://github.com/mcintyre94/orbit)

Currently it is deployed to Vercel

## Local Development

- Install dependencies: `pnpm install`
- Copy `.env.local` and set the `JUPITER_API_KEY`. You can get a [free API key here](https://portal.jup.ag)
- `pnpm vercel dev`
- API will run on localhost:3000
